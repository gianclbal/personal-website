description: A MERN web app for buying and selling listings in an online market. Users must create an account and login to buy and sell. Users can do the following actions - create and delete a listing; add and remove a listing from cart, and auction their listing. Auction feature is handled by websocket, a bidirectional communication protocol that can send the data from the client to the server or from the server to the client.
